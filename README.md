# ✨ Onde o café se mistura com magia - Coffee MoonBrew🧙‍♂️
## Todo bom feitiço começa com a organização. Mergulhe nas páginas deste projeto para desvendar seus segredos!
> Uma página mágica e responsiva para uma cafeteria temática gourmet. O projeto foi desenvolvido com o obejtivo de porporcionar uma experiência  de usuário fluída, destacando o cardápio, o storytelling da marca. 


## 🌟 Tecnologias Utilizadas 🌟
Utilizando a mais pura essência de:
- JavaScript - manipulação para interatividade
- HTML - semântica e estrutura de dados.
- Tailwind - estilização, animações e magia da página
- Google Fonts - tipografia personalizada para eforçar a identidade visual

## 🔮 Funcionalidades:
- **Design Responsivo**: Adaptado para dispositivos móveis, tablets e desktops.
- **Frontend**: uma interface intuitiva e responsiva, projetada com projetada com classes Tailwind para ser agradável em qualquer dispositivo mágico.
- **Lógica**: o motor JavaScript orquestra todas as ações, garantindo precisão na execução.
- **UX**: transições suaves e feedbakc visual claro para que o usuário saiba exatamente o que está acontecendo em cada momento.
- **Menu Interativo**: Exibição detalhada de produtos e preços.
- **Seção Sobre Nós**: Espaço dedicado à história e valores da cafeteria.
- **Formulário de Contato**: Integrado para captação de leads ou reservas.
- **Animações Suaves**: Transições em CSS para uma navegação mais moderna.
